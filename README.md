## setting

1️⃣ 터미널에서 아래 명령어를 입력하면 git을 설치합니다.

```
git --version
```

2️⃣ git clone 명령어를 실행하여 개발 환경 스크립트를 복사합니다.

```
git clone https://github.com/sjoleee/how-i-met-my-laptop.git
```

3️⃣ 개발 환경을 자동으로 설정합니다.

```
./how-i-met-my-laptop/install.sh
```
